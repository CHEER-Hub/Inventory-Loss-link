Data_Fusion Class

class Data_Fusion.Data_Fusion(cwd)

Inventory class to initialize the Inventory method. It also downloads the US county borders from <https://public.opendatasoft.com/explore/dataset/us-county-boundaries/download/?format=geojson&timezone=America/New_York&lang=en>`, with the FIPS codes and County names included, as those will be required later. All geometries are turned unto ESPG:4326, if not, throughout this module.

Parameters:

cwd (str) – Path to the working directory.

Variables:

  • cwd (str) – Stores the working directory path.

  • county_bound (GeoDataFrame) – Stores GeoDataFrame of US counties.

  • us_state_to_abbrev (list) – Stores a list of dictionary of US states to ther abbreviated names.

  • utils (Class) – Stores the utils class for future uses.

  • state_codes (list) – Stores a list of US state FIPS code.
